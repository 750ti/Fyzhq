# Fyzhq
## Fyzhq是什么
发音转换器，可以将中文的发音对应到外语发音，以达到用外语说中文的效果。

## 如何使用？
输入想转换的文字，将得到的结果复制到谷歌翻译等网站即可得到发音。
* 谷歌翻译：[https://translate.google.cn](https://translate.google.cn)
* 百度翻译：[https://fanyi.baidu.com](https://fanyi.baidu.com)


## 参与项目
